# Get More Ram
这是一个从[StosSign](https://github.com/stossy11/StosSign)分离出来的简单软件，它可以让你在没有Xcode的情况下给你的侧载软件增加内存上限。<br>
提示：这个fork已经被写的大部分都是中文了，增加了耐看图标和中文翻译<br>TIP: This fork is localized to Chinese, which added good-looking icon and Chinese localization (No English)<br>
A simple [StosSign](https://github.com/stossy11/StosSign) wrapper app that allows you to enable "Increased Memory Limit" for your sideloaded apps without using Xcode.
<br>The Chinese translation is available in this build but only Chinese!!
<br>The multi-language is processing with VSCode!!!

# 如何使用/How to use
1. 侧载这个软件 ｜ Sideload this app
2. 打开设置，登录你用于签名要增加内存上限的软件的账户 ｜ Go to settings, sign in your account that you used to sign the app you want to enable "Increased Memory Limit"
3. 打开"应用ID"页 ｜ Go to "App IDs" page
4. 点击刷新 ｜ Tap Refresh
5. 点击那个你要增加内存上限的软件包名 ｜ Tap the app you want to enable "Increased Memory Limit"
6. 点击"增加内存上限 ｜ "Tap "Add Increased Memory Limit"
7. 重新在Sidestore或者Altsotre安装这个软件 ｜ Reinstall the app from SideStore/AltStore
8. 检查你是否有更多的可使用内存 ｜ Check if you have "Increased Memory Limit"
<br>提示：一般而言，你的软件公用内存将会从4GB到6GB，你可以去设置开发者页打开HUD以进行内存监控
